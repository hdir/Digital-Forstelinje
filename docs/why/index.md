---
title: Motivasjon for arkitekturarbeid
---

Formålet med arkitekturarbeidet er å vise motivasjonen bak tiltakene og hvordan disse kan realiseres med endringer på forretning og applikasjon. Det er også muilg å vise hvilke mål som kan oppnås og verdi som kan høstes ved å gjøre ulike former for endringer i forretningsprosesser og applikasjonstjenestene som skal understøtte disse.
